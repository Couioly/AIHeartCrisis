# HeartCrisis 数据库设计文档

## 概述
本文档基于项目代码中的数据库模型设计整理，严格按照 `backend/models/` 目录下的模型文件编写。

## 数据表设计

### 1. user 表 - 用户表
**表名**：`user`  
**模型文件**：`backend/models/user.py`

| 字段名 | 数据类型 | 约束 | 备注 |
|--------|----------|------|------|
| id | int | 主键、自增 | 用户ID |
| username | String(50) | 唯一、索引、非空 | 用户名 |
| password | String(255) | 非空 | 密码哈希 |
| created_at | TIMESTAMP | 自动设置 | 创建时间 |
| updated_at | TIMESTAMP | 自动更新 | 更新时间 |

## 基类说明
**文件**：`backend/models/base.py`

所有表都继承自 `Base` 抽象基类，自动包含以下字段：
- `created_at`: 创建时间（插入时自动赋值，不修改）
- `updated_at`: 更新时间（插入/更新时自动刷新）

---

## 数据库连接配置
**文件**：`backend/models/db_conn.py`

- 使用 SQLAlchemy 异步引擎
- 异步会话工厂：`AsyncSessionLocal`
- 按日期生成 SQL 日志文件到 `logs/` 目录
